**Instrucciones**

1.  **Ejecuta el programa principal**:

    ``` bash
    python vista/main.py
    ```

2.  **Selecciona una opción del menú:**

    -   **1. Insertar libro:** agrega un nuevo registro con título,
        autor y precio.
    -   **2. Mostrar todos:** lista todos los libros guardados.\
    -   **3. Buscar por ID:** muestra la información de un libro
        específico.
    -   **4. Actualizar libro:** modifica los datos de un libro
        existente.
    -   **5. Eliminar libro:** borra un registro por su ID.\
    -   **6. Salir:** cierra el programa.

3.  **La base de datos** se guarda automáticamente en la carpeta
    `datos/libros.db`.

4.  Si deseas **probar inserciones simultáneas**, puedes ejecutar la
    función de inserción concurrente desde el controlador
    (`operaciones.py`).
